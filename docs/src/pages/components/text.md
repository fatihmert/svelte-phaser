# Text

Renders text onto the canvas.

## Usage

```example
<script>
  import { Game, Scene, Text } from 'svelte-phaser'
</script>

<Game width={400} height={400}>
  <Scene key="main">
    <Text x={100} y={180} text="hello world" color="white" fontSize="28px" />
  </Scene>
</Game>
```
